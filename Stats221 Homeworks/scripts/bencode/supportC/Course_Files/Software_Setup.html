SPSS

We will be using SPSS this semester to perform most of our statistical computations. There are two ways you can use SPSS, namely you can (1) purchase a 6 month copy to install on your personal computer, or you can (2) use VMware to access a copy of SPSS that's installed on a university computer. We recommend purchasing SPSS as it often leads to the fewest technical issues, though the VMWare option is available if you are comfortable working on a virtual machine.

Option 1: Purchase SPSS

If you decide to purchase SPSS, you can go to the onthehub.com/spss website and follow these directions:

  • After clicking "Buy Now" for students, a bunch of SPSS package options will appear on the page. Just above the first listed package there is a drop down box to select Windows or Mac. Click on whichever one applies to you. Then scroll down to "IBM® SPSS® Statistics Base GradPack 24 ... (06-Mo Rental)". This version costs $39.99 + $4.99 download fee (it may be slightly higher if they've changed their pricing). This 6 month GradPack package will give you everything you need for the semester.
  • You will need to keep close track of your purchase email. When you download SPSS and try to open it you will not be able to use it until you enter in the license number. The license number and the instructions on how to enter it into SPSS are in the purchase email. If you need further instructions, please review the file SPSS Installation Spring 2015.pdf.

Option 2: Use VMWare to access SPSS

If you decide instead to Access SPSS with VMWare, follow the instructions provided by the computer help desk:

  • PC Instructions
  • Mac Instructions

Please install any needed software on your computer they point you to, and then connect to a university computer. Note that when you connect to a university computer, you literally are working remotely on a different computer. The files on your computer are not on that computer, and vice-versa.

Open a file in SPSS

Once you have installed SPSS, please download the file MysteryInvestmentOpportunities.sav. Then open the file with SPSS. All you need to do is make sure the file opens. When it opens, you should see a column that contains years from 2011 and below, and then 5 more columns with numbers in them. You'll use this file as part of your individual preparation work for Lesson 03. Make sure you can open files with SPSS before this time comes.

Note for VMWare users

If you decided to use VMWare, then you need to realize that you may have downloaded the file above on your computer, but are trying to use SPSS on a remote computer. To fix this, one option is to first connect to the remote computer with VMWare, then open I-Learn to this page. While on the remote computer, download the file and then open it in SPSS.  You can also email yourself files to transfer documents into and out of VMWare.

Once you can open files with SPSS, mark this assignment complete.

Software Setup

Microsoft Excel

You will need a recent version of Microsoft Excel. You can use Microsoft Excel 2010 or later. A free download of Microsoft Office is available at byuistore.com. Make sure you have a working copy installed on your computer.

Wiki Settings, Data Files, and Excel Tools

Please open the Data Files web page. Select the appropriate Course type, Course code, and Instructor from the drop down menus in the upper right. Then download one of the data files and make sure it opens in your version of excel.

Please open the Excel Tools web page. Download zip file containing the three Excel files (or just download each individually), and then open one of the files to make sure you can use these sheets.

Once you have a working copy of Microsoft Excel on your computer, and have correctly set the drop down menus in the wiki, mark this assignment complete.
